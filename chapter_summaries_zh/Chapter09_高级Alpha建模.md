# 第9章 高级 Alpha 建模技术 (Advanced Alpha Modeling Techniques)

## 9.1 章节概述

本章超越"一刀切"的 alpha 模型，提出三种高级建模方法：
1. **情境建模 (Contextual Modeling)** ——按风险维度（价值/成长/盈利波动率）分割股票，构建情境化模型。
2. **行业建模 (Sector-based Modeling)**
3. **非线性效应建模 (Nonlinear Effect Modeling)**

## 9.2 收益生成方程

r_i^{t+1} = α_i + Σ b_{ik} f_k^t + ε_i^{t+1}

横截面差异 → α 因子；行业/风格差异 → β 因子；资产特异 → ε。

## 9.3 情境建模 (Contextual Modeling)

### 9.3.1 思想

把股票按某个风险维度（如 P/B 高/低）分成两个 context，分别建立 alpha 模型。理由：投资者行为差异 → 因子在不同 context 下定价机制不同。

### 9.3.2 数学

定义高/低维度的 IC：IC_H, IC_L，离散度 dis(F_H), dis(F_L)。

整体 IR：

IR = (IC̄_H + IC̄_L) / √(σ_H² + σ_L² + 2ρ_{H,L}σ_Hσ_L)

例：若 IC_L = 0（因子在低维度无效），整体 IR 反而被稀释。

### 9.3.3 最优情境权重

v_H*/v_L* ∝ (IC̄_H/σ_H² - ρ_{H,L} IC̄_L/σ_Hσ_L) / (IC̄_L/σ_L² - ρ_{H,L} IC̄_H/σ_Hσ_L)

### 9.3.4 实证示例 (Russell 1000, 1986–2004)

按三大风险维度分割：
- **价值维度**: 高 P/B vs 低 P/B
- **成长维度**: 高 growth vs 低 growth
- **盈利波动率**: 高 vs 低

结果：
- 外部融资 (EF) 因子：低价值股 IC=0.044，高价值股 IC=0.015 → 显著差异。
- 价值因子（RV）权重：低成长 22.8% vs 高成长 3.7% → 显著差异。

### 9.3.5 实证业绩

| 模型 | Alpha | IR |
|------|-------|----|
| 静态（一刀切） | 7.41% | 1.56 |
| 价值情境 | 8.53% | 1.63 |
| 成长情境 | 8.54% | 1.66 |
| 综合情境 | 8.57% | 1.72 |

### 9.3.6 模型距离测试

距离指标：d = √(Δw'·Δw / k)

通过 bootstrap + 卡方检验 → 价值与成长维度显著区别于静态模型。

## 9.4 行业 vs 情境建模

行业建模：每个行业独立模型（金融、能源、科技…）。

层次结构：行业 → 情境（双层细分）。

### 行业建模适用场景

- 行业商业模式差异极大（金融业不能用 RNOA）。
- 利率敏感性显著（金融业）。

## 9.5 非线性效应建模

### 9.5.1 案例：CAPEX

线性模型隐含"CAPEX 越低 → 收益越高"。但实证显示：
- 高 CAPEX 公司：收益最差（代理问题）。
- 中等 CAPEX 公司：收益最佳。
- 极低 CAPEX 公司：收益也较差（投资不足）。

——回报关于 CAPEX 是 **凹函数**。

### 9.5.2 解决方法

1. **多项式扩展**：F = a·CAPEX + b·CAPEX² + c·CAPEX³。
2. **条件因子**：用其他公司属性（如 ROE）来决定 CAPEX 的"合适性"。

例：CAPEX × (ROE > median) 形成"高质量增长"信号。

### 9.5.3 注意

- 多项式法易过拟合。
- 条件因子需要经济直觉支撑。

## 9.6 模型距离测试 (Model Distance Test)

测试两模型权重差异是否显著：

d = √(Δw'·Δw / k)

通过 bootstrap 重抽样 1000 次得到 d 的分布。计算卡方统计量 + p-value。

## 9.7 学习目标

- 推导情境建模的最优权重。
- 应用 Bootstrap 卡方检验比较模型。
- 区分"行业建模"和"情境建模"。
- 用条件因子建立非线性模型。
