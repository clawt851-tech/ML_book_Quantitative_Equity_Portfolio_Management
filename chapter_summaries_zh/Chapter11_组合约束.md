# 第11章 组合约束与信息比 (Portfolio Constraints and Information Ratio)

## 11.1 章节概述

本章研究"组合约束"如何影响 IR：
- 行业中性 (sector neutral)
- 多空比例 (long/short ratio)
- 仅多头 (long-only)
- 范围约束 (range constraint)

并量化"约束损失"：从理论 IR 到净 IR 的衰减程度。

## 11.2 行业中性约束 (Sector Neutral Constraint)

约束条件：每个行业内主动权重之和 = 0。

Σ_{i∈S} w_i = 0, ∀S

### 11.2.1 影响分析

最优主动权重在每个行业 S 内调整为：

w_i = (1/λ) S⁻¹(f_i - l_S β_i - μ_S)

其中 μ_S 是该行业的拉格朗日乘子。

### 11.2.2 IR 损失

行业中性损失约 5%–10%（依赖因子在行业间分布）。

## 11.3 多空比与无约束组合 (Long/Short Ratio of Unconstrained)

### 11.3.1 基准权重的对数正态分布

实证发现：S&P 500 等权重符合**对数正态分布**：

p(b) = (1/(b·σ·√2π)) · exp(-(ln b - μ)²/(2σ²))

模拟基准的算法：
1. p_i = 1 - (i-0.5)/N
2. y_i = Φ⁻¹(p_i)
3. s_i = exp(c·y_i)
4. b_i = s_i / Σ s_j

c = 浓度参数（c=0 等权重，c↑ 越集中）。

### 11.3.2 单股票多空概率

主动权重：w_i = σ_target · F_i / (√N · σ_i)

总权重：W_i = w_i + b_i ~ N(b_i, s_i²)

短仓概率：P(W_i < 0) = ∫_{-∞}^0 N(x; b_i, s_i²) dx

### 11.3.3 平均长/短仓位

S̄ = Σ E[w_i + b_i | w_i + b_i < 0]
L̄ = Σ E[w_i + b_i | w_i + b_i > 0]

公式（11.18）：

E[w_i + b_i | < 0] = -s_i/√(2π) · exp(-b_i²/(2s_i²)) + b_i · CDF(-b_i, 0, s_i)

### 11.3.4 跟踪误差与杠杆

实证：500 股票、c = 1.2、specific risk = 40%：
- TE 0.5%：long 102%，short -2%，leverage 104%
- TE 2.5%：long 131%，short -31%，leverage 162%
- TE 5.0%：long 179%，short -79%，leverage 258%

## 11.4 仅多头组合 (Long-Only Portfolios)

### 11.4.1 数学表述

最大化 f'w
s.t.
- w'Σw = σ²_target
- w'i = 0, w'B = 0
- L_i ≤ w_i ≤ U_i
- 长仓约束：w_i ≥ -b_i

### 11.4.2 Kuhn-Tucker 条件

w_i = (f_i - l_0 - l_1 b_{1i} - ... - l_K b_{Ki} + l̃_{1i} - l̃_{2i}) / (2λσ_i²)

### 11.4.3 数值求解算法（迭代）

1. 应用范围约束：w_i^new = max(min(w_i^new, U_i), L_i)
2. 更新拉格朗日乘子（基于 KKT 条件）
3. 调整跟踪误差到目标值
4. 检查约束，重复直至收敛

### 11.4.4 Transfer Coefficient

TC = corr(w^constrained, f)

约束越严，TC 越低。Long-only 典型 TC ≈ 0.7。

### 11.4.5 IR 损失

实证（500 股票，TE=3%）：

| 组合 | Net IR |
|------|--------|
| 长仓约束 (Long-Only) | 1.38 |
| 长 109% / 短 -9% | 1.48 |
| 长 121% / 短 -21% | 1.59 |
| 长 138% / 短 -38% | 1.71 |
| 无约束 | 1.77 |

——长仓限制下 IR 比无约束少约 22%。

## 11.5 长短组合与杠杆成本

### 11.5.1 杠杆成本

实际 1:1 杠杆 ≠ 自由：
- 借券利息成本 1% / 杠杆。
- 净 IR = (毛收益 - 借贷成本 - 交易成本) / 实际跟踪误差。

### 11.5.2 仿真假设

- 500 股票
- 跟踪误差 0.5%–5%
- 借贷成本 1%
- 交易成本 1% / 100% 换手率
- 预测自相关 0.25

## 11.6 大小盘风险分配

仅多头组合的额外问题：风险被迫集中在大盘股。

实证：500 股票按市值五分位分组，TE=3% 的仅多头组合：
- 最大五分位贡献 45% 风险
- 第二大 17%
- 其余三组各约 13%

——alpha 在小盘股的优势难以兑现。

## 11.7 学习目标

- 用对数正态模拟基准权重。
- 推导单股票多空仓位概率分布。
- 实现 KKT 条件下的仅多头数值算法。
- 计算 IR 损失（理论 IR vs 净 IR）。
