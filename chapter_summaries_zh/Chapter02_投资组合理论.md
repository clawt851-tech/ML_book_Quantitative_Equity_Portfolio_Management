# 第2章 投资组合理论 (Portfolio Theory)

## 2.1 章节概述

本章系统讲解 Markowitz 均值-方差理论及其推广（含/不含现金、主动组合、Beta-中性组合），并引入 CAPM 框架。它是后续所有"最优化"内容的数学基础。

## 2.2 核心要点

### 2.2.1 投资收益的分布 (Distributions of Investment Returns)

- 单期收益向量 r ~ N(μ, Σ)。
- 组合收益 r_p = w'r 服从 N(w'μ, w'Σw)。
- 正态分布缺陷：(a) 尾部偏薄；(b) 多期复合不再正态；(c) 不能直接刻画对数收益。
- 实证观察：股票收益常具偏度与厚尾（fat tails）。

### 2.2.2 相关性与分散化

- ρ_{1,2} = σ_{12} / (σ_1 σ_2)，|ρ| ≤ 1。
- 两资产组合方差： σ_p² = w_1²σ_1² + 2ρw_1w_2σ_1σ_2 + w_2²σ_2²。
- 极端：ρ = 1（无分散收益），ρ = -1（最大分散收益）。
- N 资产等权且不相关时：σ_p = σ/√N。

### 2.2.3 最优组合 (Optimal Portfolios)

#### (a) 最小方差组合 (Minimum Variance)

minimize ½ w'Σw  s.t. w'i = 1。

解：w*_min = Σ⁻¹i / (i'Σ⁻¹i)。

#### (b) 含现金的均值-方差最优组合

maximize w_0 r_f + w'f - ½λ(w'Σw)  s.t. w_0 + w'i = 1。

解：w* = (1/λ) Σ⁻¹ f_e， 其中 f_e = f - r_f i。

#### (c) 不含现金（全额投资）

w* = Σ⁻¹i/(i'Σ⁻¹i) + (1/λ) [(i'Σ⁻¹i)Σ⁻¹f - (i'Σ⁻¹f)Σ⁻¹i] / (i'Σ⁻¹i)。

#### (d) 主动均值-方差最优化 (Active MV)

w = b + a，b 为基准权重，a 为主动权重。

maximize a'f - ½λ(a'Σa)  s.t. a'i = 0。

解：a* = (1/λ) [(i'Σ⁻¹i)Σ⁻¹f - (i'Σ⁻¹f)Σ⁻¹i] / (i'Σ⁻¹i)。

**重要洞察**：主动权重与基准无关——只要风险模型与预测一致，可"port"到任何基准（portable alpha）。

### 2.2.4 资本资产定价模型 (CAPM)

- r_i = r_f + β_i(r_M - r_f) + ε_i。
- β_i = cov(r_i, r_M)/σ²_M。
- 协方差矩阵分解：Σ = ββ'σ²_M + diag(θ_i²) = ββ'σ²_M + S。
- κ = Σ σ²_M β_i² / θ_i² 衡量系统/特异方差比，对 N 越大，组合 beta 趋于零。

### 2.2.5 CAPM 下的最优组合

- 部分解（仅按特异风险）：w_0 = (1/λ) S⁻¹ f_e。
- 完整解：w* = w_0 - σ²_M β_0 / (1+κ) β_s。
- 单股票最优权重：w_i* = w_{0,i} - cov(r_i, r_{w0}) / [(1+κ) θ_i²]。

### 2.2.6 Beta-中性组合 (Beta-Neutral Portfolios)

- 约束 w'β = 0。
- 解：w* = (1/λ) S⁻¹ (f - lβ)，l = f'S⁻¹β / β'S⁻¹β。
- 单股票：w_i = (1/λ)(f_i - lβ_i)/θ_i²。

### 2.2.7 特征组合 (Characteristic Portfolios)

- 与某特征 c 相关的最优组合：w_c ∝ Σ⁻¹c/(c'Σ⁻¹c)。

## 2.3 关键公式速查

| 概念 | 公式 |
|------|------|
| 最小方差组合 | w = Σ⁻¹i/(i'Σ⁻¹i) |
| 均值-方差(含现金) | w = (1/λ) Σ⁻¹ f_e |
| 均值-方差(全额投资) | w = w_min + (1/λ) Σ⁻¹(f - lᵢ)，l = i'Σ⁻¹f/(i'Σ⁻¹i) |
| Beta-中性组合 | w = (1/λ) S⁻¹(f - lβ) |
| CAPM 期望收益 | E(r_i) = r_f + β_i(r_M - r_f) |

## 2.4 学习目标

- 掌握 Markowitz 均值-方差最优解推导。
- 理解 portable alpha：主动权重与基准无关。
- 解释 CAPM 协方差矩阵的"系统+特异"结构。
- 推导 Beta-中性约束下的最优权重。
