# 第12章 交易成本与组合实施 (Transaction Costs and Portfolio Implementation)

## 12.1 章节概述

本章把 alpha 从"纸面"转化为"实盘"。两大主题：
1. **带交易成本的最优组合 (Optimal Portfolio with Transaction Costs)**
2. **最优交易策略 (Optimal Trading Strategies)**

## 12.2 交易成本的组成

| 类型 | 描述 |
|------|------|
| 委员佣金 (Commission) | 经纪商收费，固定或随交易量计算 |
| 买卖价差 (Bid-Ask Spread) | 流动性提供者补偿 |
| 市场冲击 (Market Impact) | 大单冲击价格 |
| 时机成本 (Timing Cost) | 延迟成交带来的机会成本 |

### 12.2.1 数学形式

线性成本：θ·|Δw|
平方成本：ψ·(Δw)²
混合：θ·|Δw| + ψ·(Δw)²

## 12.3 单资产带交易成本最优化

### 12.3.1 平方成本

minimize: -f·w + ½λσ²w² + ψ(w-w_0)²

解：w* = (f + 2ψ w_0) / (λσ² + 2ψ)

### 12.3.2 线性成本

minimize: -f·w + ½λσ²w²
s.t. cost = θ·|w-w_0|

KKT 条件：
- 若 w > w_0：调整后预测 f-θ
- 若 w < w_0：调整后预测 f+θ

存在"无交易区间"：当 (f-θ)/λσ² ≤ w_0 ≤ (f+θ)/λσ²，最优解就是 w_0。

例：f=15%, σ=15%, λ=10, w_0=50%
- θ=0：w*=66.7%
- θ=0.01：w*=62.2%
- θ=0.02：w*=57.8%
- θ≥0.0375：w*=50%（不交易）

## 12.4 多资产带交易成本

### 12.4.1 二次型成本

maximize: f'w - ½λw'Σw - (Δw)'Ψ(Δw)

解析解：w* = (λΣ + 2Ψ)⁻¹ (f + 2Ψw_0)

### 12.4.2 线性成本（数值法）

maximize: f'w - ½λw'Σw - θ'·|Δw|

引入买卖向量：
- w_B ≥ 0（买入向量）
- w_S ≥ 0（卖出向量）
- w = w_0 + w_B - w_S
- |Δw| = w_B + w_S（互斥时）

转化为标准 QP：

maximize: F'W - ½λW'Σ_2W

W = [w_B; w_S], F = [f - λΣw_0 - θ; -f + λΣw_0 - θ], Σ_2 = [Σ -Σ; -Σ Σ]

约束：W ≥ 0；可加 dollar neutral, range 等约束。

### 12.4.3 实证示例（20 股票）

- 预测与初始权重相关性仅 0.48
- 优化后单边换手率约 36%
- Gross alpha 8.5%, net alpha 7.0%
- 限制换手率到 20%：net alpha 6.5%

## 12.5 最优交易策略

### 12.5.1 单股票

把交易分散在 [0, T] 内执行。h(t) 表示截至 t 的累计执行比例（h(0)=0, h(T)=1）。

目标函数：

J = ∫_0^T f·Δw·[h(t)-1] dt - ½λσ²(Δw)²∫_0^T [h(t)-1]² dt
    - c·|Δw|·T - ψ(Δw)²∫_0^T [ḣ(t)]² dt

四项含义：
1. 预期收益（来自未实现部分）
2. 未实现仓位的方差（实施风险）
3. 固定成本随时间累积
4. 市场冲击成本

### 12.5.2 关键 ODE

ḧ - g²h = -s - g²
g² = λσ²/(2ψ)
s = f_w/(2ψ)

边界条件：h(0)=0, h(T)=1

### 12.5.3 三种特例

**Case I: s=g=0（无 forecast，无 risk aversion）**

h(t) = t/T → 匀速交易（恒定速度 1/T）

**Case II: g=0（无 risk aversion，有 forecast）**

h(t) = t/T + (s/2)·t·(T-t)

s>0: 先快后慢；s<0: 先慢后快。

**Case III: 一般情形 (g>0)**

h(t) = [(1+s/g²)cosh(gT)-s/g²] / sinh(gT) · sinh(gt) - (1+s/g²)·[cosh(gt)-1]

### 12.5.4 最优交易时长

最优 T 使总成本（固定+冲击+实施风险）最小：

- 简化情形 (s=0)：T* = √(ψ/c_w)

——市场冲击成本 ψ 越高，应执行得越久；固定成本 c_w 越高，应执行得越快。

### 12.5.5 风险与成本的有效前沿

在 implementation cost vs implementation risk 平面，不同 λ 给出一条曲线，类似均值-方差有效前沿。投资经理依风险偏好选择。

## 12.6 组合层面的交易策略

- 互相关性：同方向交易的股票同步执行可减少系统暴露。
- 净市场暴露平衡：买卖应该在 dollar 上平衡。
- 流动性差异：illiquid 股票需要更长执行窗。

## 12.7 学习目标

- 推导带平方/线性交易成本的最优权重。
- 把线性成本问题化为标准 QP（buys/sells split）。
- 用变分法解最优交易路径。
- 理解风险-成本权衡，构建实施有效前沿。
