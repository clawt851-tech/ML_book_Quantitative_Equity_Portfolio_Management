# 第8章 组合换手率与最优 Alpha 模型 (Portfolio Turnover and Optimal Alpha Model)

## 8.1 章节概述

本章把"换手率约束"引入 alpha 模型最优化。核心结论：

**预测自相关 ρ_f 越高，换手率越低。** 用滞后预测（lagged forecasts）作为模型组件可大幅降低换手，即便单期 IC 略低，多期净收益依然可能更高。

## 8.2 被动组合漂移 (Passive Drift)

### 8.2.1 定义

固定权重组合在不交易的情况下，权重会因价格变化而漂移。

漂移引发的换手：

T ≈ d / [(1+r̄)·√(2π)]

d = 横截面收益离散度，r̄ = 平均收益。

### 8.2.2 一般公式

带杠杆 L 的组合：

T ≈ L · d/(2(1+r_p)) · [1 + (Δr)²/(2d²)]

## 8.3 预测变化引致的换手

### 8.3.1 关键公式

无约束多空组合：

T = √(N/π) · σ_{model} · √(1 - ρ_f) · E(1/σ)

四个驱动因素：
- 跟踪误差 σ_{model} 越高，换手越高。
- 股票数 N 越大，换手 ∝ √N。
- 预测自相关 ρ_f 越高（越接近 1），换手越低。
- 特异风险 σ 越高，换手越低（相同 z-score 对应的权重小）。

### 8.3.2 杠杆与换手

L = √(2N/π) · σ_{model} · E(1/σ)

T = L · √(1 - ρ_f) / √2

——换手率与杠杆成正比。

### 8.3.3 各类因子的预测自相关

| 类别 | 因子 | ρ_f |
|------|------|-----|
| 价值 | E2PFY0 | 0.96 |
| 价值 | B2P | 0.93 |
| 价值 | CFO2EV | 0.84 |
| 质量 | RNOA | 0.89 |
| 质量 | XF | 0.76 |
| 质量 | NCOinc | 0.80 |
| 动量 | EarnRev9 | 0.64 |
| 动量 | Ret9Monx1 | 0.60 |
| 动量 | LtgRev9 | 0.37 |

价值最稳定，动量最高变。

## 8.4 复合预测的换手

### 8.4.1 自相关

ρ_{f_c} = (v_1²ρ_{11}^{t,t+1} + v_2²ρ_{22}^{t,t+1} + v_1v_2(ρ_{12}^{t,t+1} + ρ_{21}^{t,t+1})) / (v_1² + v_2² + 2v_1v_2ρ_{12}^{t,t})

### 8.4.2 移动平均 (Moving Averages)

L 阶移动平均：F^ma_t = Σ v_l F^{t-l}

效果：增加序列自相关 → 降低换手。但需衡量信息衰减。

## 8.5 信息时域与滞后预测 (Lagged & Horizon IC)

### 8.5.1 滞后 IC

IC^l_t = corr(F_t, R_{t+l})

通常 IC 随滞后衰减。

### 8.5.2 视野 IC (Horizon IC)

IC^h_t = corr(F_t, R_{t,t+h})

关系式：IC^l_t ≈ avg(IC) · √(l+1)

### 8.5.3 有效 IC vs 有效换手

效能比：

Q_{IC,T} = IC^l_t · √(l+1) / √(1-ρ(l+1))

——单位换手获得的 IC。

## 8.6 带换手约束的最优 Alpha 模型

最大化 IR 受 ρ_{fc,ma} = ρ_t 等式约束：

maximize IR = v'IC̄/√(v'Σ_IC v)
s.t. ρ_{fc,ma} = v'D_4 v / v'C_4 v = ρ_t

（C_4, D_4 是滞后/同期相关矩阵；详见章末附录）

无解析解，需数值优化。

## 8.7 关键结论

- 即使滞后 IC 较弱，使用滞后预测仍可降低换手 → 净 IR 更高。
- 价值因子的滞后预测信息衰减慢，可重权使用。
- 动量因子滞后衰减快，应较少滞后。

## 8.8 学习目标

- 推导 T = √(N/π)·σ_model·√(1-ρ_f)·E(1/σ)。
- 解释为何高自相关 → 低换手。
- 应用移动平均降低换手率。
- 在带换手约束下数值求解最优权重。
